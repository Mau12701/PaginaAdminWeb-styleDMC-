HTML y CSS

Glosario de HTML y CSS

Estructura del Documento

El documento HTML está estructurado con las siguientes etiquetas básicas:

EtiquetaDescripción
<html>Define el inicio del documento HTML.
<head>Contiene metadatos y enlaces a recursos.
<body>Contiene el contenido visible de la página.
<header> Representa la cabecera de la página o de una sección. Suele incluir el título principal (h1) la navegación principal (nav).
<!DOCTYPE html>tipo de documento y la versión de HTML que se está utilizando (en este caso, HTML5).

La estructura básica de un documento HTML es esencial para que los navegadores puedan interpretar y mostrar correctamente el contenido.

Etiquetas de Contenido y Texto

Se utilizan para definir texto y su estructura:

EtiquetaDescripción
<p>Define un párrafo.
<h1> - <h6>Definen encabezados de diferentes niveles.
<span>Resalta partes de un texto sin afectar su estructura.
<strong>Resalta texto en negrita, indicando importancia.
<em>Resalta texto en cursiva, indicando énfasis.

Estas etiquetas ayudan a organizar y dar formato al texto, mejorando la legibilidad y la estructura del contenido.

Listas

Existen tres tipos de listas en HTML:

EtiquetaDescripción
<ul>Lista desordenada.
<ol>Lista ordenada.
<dl>Lista de definición.

Las listas son útiles para organizar información de manera estructurada, ya sea en forma de viñetas, números o términos y definiciones.

Enlaces y Navegación

Los enlaces permiten la navegación entre páginas.

EtiquetaDescripción
<a>Define un hipervínculo.
<nav>Define una sección de navegación.

Los enlaces son fundamentales para la navegación web, permitiendo a los usuarios moverse entre diferentes páginas y recursos.

Imágenes y Multimedia

Elementos para insertar imágenes, audio y video:

EtiquetaDescripción
<img>Inserta una imagen.
<video>Inserta un video.
<audio>Inserta audio.

El uso de imágenes y multimedia enriquece el contenido de las páginas web, haciéndolas más atractivas e interactivas.

Tablas

Usadas para mostrar información en formato tabular.

EtiquetaDescripción
<table>Define una tabla.
<tr>Define una fila.
<th>Define una celda de encabezado.
<td>Define una celda de datos.

Las tablas son ideales para presentar datos de manera organizada y fácil de entender.

Formularios y Entradas de Datos

Permiten la interacción con el usuario mediante campos de entrada.

EtiquetaDescripción
<form>Define un formulario.
<input>Campo de entrada de datos.
<button>Botón interactivo.

Los formularios son esenciales para recopilar información de los usuarios, como datos de contacto, comentarios y más.

Elementos Semánticos y de Sección

Mejoran la estructura y accesibilidad del documento.

EtiquetaDescripción
<header>Define la cabecera de una página o sección.
<article>Define un contenido independiente.
<footer>Define el pie de página.

Los elementos semánticos ayudan a los motores de búsqueda y a las tecnologías de asistencia a entender mejor la estructura del contenido.

Elementos Interactivos y Gráficos

Elementos que permiten interacción y gráficos.

EtiquetaDescripción
<canvas>Área de dibujo para gráficos.
<svg>Define gráficos vectoriales.

Estos elementos permiten crear gráficos y animaciones interactivas, mejorando la experiencia del usuario.

Elementos de Inclusión y Embebidos

Se usan para incrustar contenido externo.

EtiquetaDescripción
<iframe>Incrusta otra página web.
<embed>Incorpora contenido multimedia.

Estos elementos permiten integrar contenido de otras fuentes, como mapas, videos y más, directamente en la página web.

Meta-Información y Recursos

Define información sobre el documento.

EtiquetaDescripción
<meta>Define metadatos.
<link>Enlaza recursos externos.

Los metadatos son importantes para el SEO y la configuración del documento, como la descripción, autor y palabras clave.

Etiquetas Obsoletas

Etiquetas que no se recomiendan en HTML5.

EtiquetaAlternativa
<font>Usar CSS para estilos.
<center>Usar CSS con `text-align`.

Es importante utilizar las prácticas modernas de HTML5 y CSS para mantener el código actualizado y compatible con los estándares actuales.
